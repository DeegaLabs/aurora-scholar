# Logo Ideas - Aurora Scholar

> Visual concepts and design guidelines

## Visual Concepts

### Concept 1: Aurora + Book

**Description:**
- Aurora (sunrise) overlaid on an open book
- Lines representing light rays coming from the book
- Symbolizes illuminated knowledge and authenticity

**Elements:**
- Rising sun (aurora) in orange/yellow gradient
- Stylized book in dark blue
- Light lines connecting aurora and book
- Modern and clean typography

### Concept 2: Aurora + Blockchain

**Description:**
- Stylized aurora as light waves
- Blockchain blocks integrated into aurora
- Represents immutable and transparent knowledge

**Elements:**
- Aurora waves in gradient
- Hexagonal blocks representing blockchain
- Visual connection between elements
- Tech-friendly typography

### Concept 3: Minimalist - Stylized Letter A

**Description:**
- Letter "A" stylized as aurora
- Ascending lines representing light
- Simple, memorable, scalable

**Elements:**
- "A" in custom font
- Light lines coming from base
- Colors: orange/yellow gradient
- Monochrome version for black/white use

### Concept 4: Aurora + Verification Shield

**Description:**
- Aurora as background
- Verification shield overlaid
- Represents authenticity and protection

**Elements:**
- Aurora in soft gradient
- Shield with checkmark
- "Scholar" typography below
- Colors: blue (trust) + orange (aurora)

## Suggested Color Palette

### Main Palette

**Aurora Orange (Primary)**
- Hex: `#FF6B35`
- RGB: `rgb(255, 107, 53)`
- Use: Main elements, CTAs, highlights

**Aurora Yellow (Secondary)**
- Hex: `#FFD23F`
- RGB: `rgb(255, 210, 63)`
- Use: Accents, gradients, light elements

**Scholar Blue (Tertiary)**
- Hex: `#1E3A8A`
- RGB: `rgb(30, 58, 138)`
- Use: Text, academic elements, contrast

**Dark Navy (Dark Neutral)**
- Hex: `#0F172A`
- RGB: `rgb(15, 23, 42)`
- Use: Dark backgrounds, text in light mode

**Light Gray (Light Neutral)**
- Hex: `#F8FAFC`
- RGB: `rgb(248, 250, 252)`
- Use: Light backgrounds, white spaces

**Aurora Gradient**
```
#FF6B35 → #FFD23F → #FF6B35

Use: Highlight elements, backgrounds

**Scholar Gradient**
```

Use: Academic elements, sections

## Typography

### Primary: Inter or System Font Stack
- **Use**: Interface, texts, UI
- **Characteristics**: Readable, modern, neutral
- **Fallback**: `-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`

### Secondary: Academic Serif (Optional)
- **Use**: Titles, brand elements
- **Suggestions**: 
  - Playfair Display (elegant)
  - Lora (academic)
  - Merriweather (traditional)

### Logo: Custom or Display Font
- **Characteristics**: Unique, memorable, scalable
- **Suggestions**:
  - Montserrat (modern)
  - Poppins (friendly)
  - Space Grotesk (tech-friendly)

## Design References

### Visual Style
- **Minimalist but welcoming**: Not too corporate
- **Modern but reliable**: Tech without being experimental
- **Academic but accessible**: Serious without being elitist

### Inspirations
- **Notion**: Cleanliness and organization
- **Obsidian**: Focus on knowledge
- **Medium**: Readability and elegance
- **GitHub**: Tech-friendly and professional

### Elements to Avoid
- Too corporate (e.g., IBM, Microsoft)
- Too childish (e.g., very vibrant colors)
- Too technical (e.g., code symbols)
- Too traditional academic (e.g., coats of arms, seals)

## Usage Guidelines

### Logo Versions

**Full Version (Horizontal)**
- Logo + "Aurora Scholar" text
- Use: Headers, presentations, documentation

**Icon Version**
- Symbol/icon only
- Use: Favicon, app icon, small spaces

**Vertical Version**
- Logo above text
- Use: Cards, printed materials

**Monochrome Version**
- Black or white
- Use: Colored backgrounds, printing

### Minimum Sizes

- **Favicon**: 32x32px
- **App Icon**: 512x512px
- **Header**: Minimum 120px width
- **Presentation**: Minimum 200px width

### Spacing

- **Breathing room**: Minimum 20% of logo size
- **Don't place**: On complex images, very colored backgrounds
- **Always maintain**: Original proportions, correct colors

### Background Colors

**Light Backgrounds**
- Use full color version
- Or dark version (Scholar Blue)

**Dark Backgrounds**
- Use light version (white/yellow)
- Or version with adequate contrast

**Colored Backgrounds**
- Use monochrome version
- Ensure minimum contrast 4.5:1
